With less than 10 active installs, Fakeblock is the OC WordPress community's privacy software plugin that's also anti-piracy.


== Description ==

Fakeblock is a unique maintenance-mode plugin.

When Fakeblock's "Privacy Mode" is enabled, visitors to your site must pass a "rhythm" test in order to gain entry to your website.

Want to see Fakeblock in action? See a live demo at [fakeblock.us](http://fakeblock.us/)!


== Installation ==

1. Install the plugin.
2. Activate it.
3. Enable privacy and anti-piracy modes under the Fakeblock menu item.
